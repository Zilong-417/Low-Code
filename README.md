# low-code低代码平台
### 项目概述：
基于vue3+element-plus开发的低代码平台<br>
支持采用图形化拖拽的方式，配置参数来生成JSON，通过JSON渲染可JSONSchema描述的页面。
### 在线demo：
https://zilong-417.github.io/Low-Code
### 技术要点分析
https://juejin.cn/post/7134722740187037732
### 安装
```
yarn install
```

### 运行
```
yarn serve
```

### 打包
```
yarn build
```
